Exercícios de fixação

Curso HTML - Aula 01

Exercício 1

  • Crie um repositório no Github chamado html.
  • Clone o repositório para sua máquina (pode baixar o arquivo .zip ou usar o comando no terminal: "git clone endereço-do-repositório").
  • Agora no seu computador, dentro da pasta do projeto que você clonou, crie uma pasta chamada aula_01, abra a pasta e siga os passos a seguir dentro dela.
  • Adicione um arquivo chamado index.html
  • Transcreva (não copie e cole) tudo do arquivo index.html da aula 01.

Exercício 2

  • Crie um novo arquivo chamado exercicio_02.html
  • Escreva um programa que imprima na tela a frase "Olá, Mundo!"

Exercício 3

  • Crie um novo arquivo chamado exercicio_03.html com as tags básicas do html (html,head,body)
  • Adicione Título, subtítulo e um texto.
  • Adicione um link âncora a sua página.
  • Adicione um "href" no seu link para ele direcionar para o site da "acadix.com.br"

Exercício 4

  • Crie um novo arquivo chamado exercicio_04.html com as tags básicas do html (html,head,body)
  • Utilize a tag <title> (dentro da tag <head>) para dar um título a sua página.
  • Adicione uma lista não ordenada com 3 itens
  • Adicione uma lista ordenada com 3 itens
  • Adicione um título a página e um subtítulo para cada lista
  • Utilizando tags <div> separe o seu código em blocos
  • Nomeie cada um dos novos blocos de código utilizando a propriedade "name"
